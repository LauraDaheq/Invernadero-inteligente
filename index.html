🌿 Invernadero Inteligente

📘 Introducción al Proyecto

Este proyecto nace como respuesta a la dificultad de mantener condiciones óptimas para la germinación y crecimiento de plantas aromáticas en interiores. Las variaciones de temperatura, humedad y luz afectan negativamente el desarrollo de especies sensibles.

La solución es un invernadero automatizado con sensores y controladores que miden variables clave y actúan en tiempo real. Además, incluye una aplicación web que permite al usuario monitorear el entorno y controlar el riego y la iluminación desde cualquier lugar.
