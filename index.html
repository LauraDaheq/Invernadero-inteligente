🌿 Invernadero Inteligente

📘 Proyecto

Este proyecto nace como respuesta a la dificultad de mantener condiciones óptimas para la germinación y crecimiento de plantas aromáticas en interiores.

Se trata de un invernadero automatizado con sensores, actuadores y una aplicación web que permite monitorear y controlar las variables ambientales en tiempo real.
