# 🌿 Smart Greenhouse IoT

An Internet of Things (IoT) project for monitoring and controlling an automated greenhouse through a web application.

The system collects environmental data from sensors, displays real-time information using ThingSpeak, and allows remote control of irrigation and lighting systems.

---

## 📖 Overview

The Smart Greenhouse was designed to automate plant care by continuously monitoring environmental conditions and allowing remote interaction with the greenhouse.

The web interface provides real-time visualization of sensor data and enables users to control irrigation and lighting from any device connected to the Internet.

## 📊 Dashboard

The dashboard displays:

- Temperature
- Air Humidity
- Soil Moisture
- Water Tank Level

Live charts are embedded from ThingSpeak and updated automatically.

---

## 🚿 Remote Control

The application allows users to remotely:

- Activate irrigation
- Stop irrigation
- Turn lights on
- Turn lights off

---

## 🔔 Alert System

Automatic alerts are generated when:

- Water level is below 20%
- Temperature exceeds 28°C

Browser notifications and an alert history improve monitoring and response.
